About

Don Swaim Collection

700 interviews and 2534 Book Beat broadcast features covering the US literary scene from WCBS in New York from 1983-1993.

  • Digital collection page
  • CONTENTdm collection alias: donswaim

Description

Normalized Data

normal/donswaim_broadcasts.json
Normalized Book Beat broadcast metadata as JSON, including full-text `Transcript`
normal/donswaim_broadcasts.tsv
Normalized Book Beat broadcast metadata as tab-seperated values, with `Transcript File Name` instead of full text transcripts
normal/donswaim_interviews.json
Normalized Don Swaim interview metadata as JSON
normal/donswaim_interviews.tsv
Normalized Don Swaim interview metadata as tab-seperated values

Element Computer-Generated Terms
CONTENTdm field nick comput
CONTENTdm system property No
Required No
DC Mapping None
Normalized No
Description A list of named entities from transcripts generated by machine learning models.
Encoding Names are seperated by semicolons
Notes Names derived from Book Beat broadcast transcripts were taken from the union of names reported by Python Natural Language Processing libraries spaCy’s en_core_web_sm and NLTK’s ne_chunk in the Fall of 2019.
Examples
  • American; Benny Goodman; Dinah; Duke Ellington; Ella Fitzgerald; Ethel Waters; Europe; George Shearing; Joe Stafford; Joe Williams; Louis Armstrong; Louis' Children; New Orleans; New York City; Warmer
  • A Little Bit Married; Broadway; Carol Burnett; I'm; Mary Hartman; New York; Putnam; Sheila Levine; The Main Event
  • American; English; Hebrew; Israeli; Land of Israel; Russian; Soviet; The Island of Crimea; U.S.; Vassily Aksyonov
